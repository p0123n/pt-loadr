rt-loadr
========

Скрипт для загрузки последних версий radio-t. При первом запуске из пустой папки
скачает 3 последние выпуска.

Далее "смотрит" что уже загружено и докачивает до последнего доступного подкаста.

(!) УДАЛЯЕТ старые версии подкастов. По умолчанию хранит последние 3 файла.

Поставить в крон можно как-то так (в 5 утра, каждое воскресенье):
```
5 5 * * Sun cd /your/path/to/radiot/files ; ./rtloadr.py
```
